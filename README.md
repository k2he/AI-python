# Getting Started

## Postman Collection
You can find the Postman collection for testing the API endpoints under /docs folder

## Start Application
1. Start mongoDB: docker compose up -d
2. Stop the DB: docker compose stop -v
3. Download packages: uv sync
4. Start FastAPI server: uv run uvicorn app.main:app
